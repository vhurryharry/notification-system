# Setup

## Setup Backend

- `yarn`
- `yarn setupDB` (This will create the database file and seed the initial data.)

For production:

- `yarn build`
- `yarn start`

For development:

- `yarn dev`

## Setup Frontend

Frontend is currently setup to use local backend running at 4040 port, which is the default port configured for the backend.

- `cd app`
- `yarn`
- `yarn start`
